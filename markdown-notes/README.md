# Markdown Notes

This folder is a Markdown mirror of the original notebook collection.

The current files were exported from `.ipynb` notebooks using:

```bash
python3 tools/export_notebooks_to_markdown.py
```

## Status

These files are mirrors, not yet polished canonical sources.

Their front matter records:

```yaml
original_human_author: B67687
conversion: exported from original notebook
canonical_status: markdown-mirror
```

This means the explanation content came from the original human-authored notebooks, while the Markdown file is a format conversion.

## Future Direction

The long-term direction is:

```text
notebooks as legacy archive
markdown as durable source
optional notebooks as generated/executable companions
```

Do not treat conversion alone as completion. Each important note should still be polished so the explanation becomes simpler, cleaner, and more fundamental.
